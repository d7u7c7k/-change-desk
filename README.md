# 자리 바꾸기 change-desk
학교에서 쓸 수 있는 자리 바꾸기 프로그램을 웹으로 만들어 보았습니다.
I made this program which can change desk in the school.

## 사용방법 (How to use)

###1.자리생성(create desk)
자리 개수를 정하고, 아래 생성버튼을 누르면 차례대로 자리가 생성됩니다. 또한 직접 자리를 옮길 수 있습니다.
Set the number of desks, and click "자리 생성" to create desks. Also you can move the desk by yourself.
###2.자리섞기(random desk)
랜덤 자리 배치 버튼을 누르면 생성된 자리들의 번호가 랜덤으로 배치됩니다.
when you click "전체 자리 랜덤 배치", desks will be moved randomly.
###3.시점(vision)
학생 시점과 선생님 시점을 변경할 수 있습니다.
"학생 시점" : Students vision, "선생님 시점" : Teacher vision
###4.인쇄(printer)
인쇄버튼을 누를시, 선생님의 시점으로 자리표를 인쇄 할 수 있습니다.
If you want to print this, click "선생님 시점으로 인쇄" button.
